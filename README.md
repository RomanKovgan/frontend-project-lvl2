## GenDiff
GenDiff - программа, которая показывает разницу между двумя структурами данных, которая поддерживает два формата: json и yaml.
Для установки необхожимо:
1. Node.js v >= 14;
2. Склонировать репозиторий локально;

## Setup
```shell
$ make install
```
## Run tests
```sh
make test
```
